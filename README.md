# 🚀 InterviewAI

AI powered interview preparation platform that generates intelligent interview answers using Generative AI.

---

## 🛠 Tech Stack

Frontend

* React.js
* Vite

Backend

* Node.js
* Express.js

Database

* MongoDB

AI

* Generative AI API

---

## 📂 Project Structure

Backend → Server and API
Frontend → User Interface

---

## ⚙️ How to Run Project

### Backend

cd Backend
npm install
npm run dev

### Frontend

cd Frontend
npm install
npm run dev
